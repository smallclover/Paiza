## Note
+ 1.所有的示例代码仅供参考（这么简单好像有人看似得_(:з」∠)_），另外，在看解答之间最好先自己做~
+ 2.所做的题目均来自[Paiza](https://paiza.jp)(该网站是日文的，如果有小伙伴看不懂，可以通过邮箱询问)
+ 3.所有的类名均以题目的编码命名，使用时根据名字查找即可
+ 4.因为刚开始，所以题目简单，随着等级提高，题目越来越难
+ 5.好像暂时就这些了
